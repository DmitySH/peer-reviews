ВНИМАНИЕ!

При компиляции с открытым окном MainWindow будут выскакивать Message Boxes, потому что он пытается десериализовать склад
в дизайнере. Это не ошибка. Чтобы убедиться можно компилировать с другим открытым в VS файлом (не окно).

Отчет CSV пригоден для чтения в Excel. Например так (Excel): Выбираем первый столбец полностью -> данные -> текст по столбцам -> далее
-> отмечаем запятую -> далее -> готово

Так же была штука, что из-за .NET5 (на коем написан проект) он на некоторых устройствах не будет компилировать вообще.
Просто нажимаем запустить последний исправный вариант и все работает.

Не недо трогать файлы внутри самого проекта =))) очевидно, так можно любую программу сломать.

И при создании пользователя корректрные данные как раз введены(как образец).
